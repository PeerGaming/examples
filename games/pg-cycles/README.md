Light Cycles
============

A simple version of a lightcycle race (see [Tron](https://en.wikipedia.org/wiki/Light_Cycles#Light_cycles)).





/**
 *  LightCycle
 *  ==========

 *  - crash into other barrier

 *
 *  - for every field you moved - let a barrier create behind
